단말기 내부 프로그램 장애 해결

단말기 내부 프로그램이 되지 않을 때 조치 방법입니다.

📌 주요 증상
  • 단말기 프로그램 접속이 안됨
  • 단말기 터치해도 동작이 안됨
  • 단말기 시간 설정이 안맞음
  • 단말기 화면이 너무 밝거나 어두움
🔍 가능 원인
  • 프로그램 일시적인 오류
  • 단말기 발열에 의한 오류
  • 재부팅 과정에서에 의한 오류
  • 전 근무자에 의한 설정에 대한 불편함
🛠 조치 방법
  1. 단말기 재부팅
    • 단말기 강제로 껐다가 킨후 확인할것
    • 전원 버튼을 (7~15초)이상 눌러서 강제로 꺼지는지 확인할것
    • 같은 방식으로 전원 버튼을 (7~15초)이상 눌러서 강제로 켜지는지 확인할것
    • 위 방식으로 했는데도 안된다면 전원 버튼을 다시 한번 눌러볼 것
    • 그래도 안된다면 배터리를 탈부착후 다시 한번 시도 할것
    • 위 과정을 통해 (터치 장애, 화면 멈춤)증상이 해결되는것을 확인
  2. 시간 설정
    • 시작 -> 설정 -> 시간 및 언어 -> 날짜 및 시간 -> 날짜 및 시간 변경 -> 해당 날짜 및 시간 설정후 확인
  3. 화면이 너무 어둡거나 밝음
    • 시작 -> 설정 -> 시스템 -> 중간에 화면 밝기 조절이 있다
  4. 위 방법으로도 조치가 안될시 조치사항
    • 통제실에 연락후 조치
